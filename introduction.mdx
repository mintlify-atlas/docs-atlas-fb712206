    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your portfolio site running in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the project to your local machine:
      
      ```bash
      git clone https://github.com/ta93abe/me.git
      cd me
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install the required packages using pnpm:
      
      ```bash
      pnpm install
      ```
      
      <Note>This project uses pnpm as the package manager. Make sure you have it installed globally.</Note>
    </Step>
    
    <Step title="Start the development server">
      Launch the local development server:
      
      ```bash
      pnpm dev
      ```
      
      Your site will be available at `http://localhost:4321`
    </Step>
    
    <Step title="Create your first content">
      Add a new blog post in `src/content/blog/`:
      
      ```md
      ---
      title: "My First Post"
      date: 2026-03-04
      excerpt: "This is my first blog post"
      tags: ["intro"]
      ---

      # Hello World!

      This is my first post using Astro.
      ```
    </Step>
  </Steps>
